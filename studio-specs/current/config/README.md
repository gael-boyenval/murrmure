# Configuration shell — retired

The Configure UI and `/setup` wizard are **retired** in Murrmure v2.

- **Admin:** CLI (`mrmr space`, `mrmr grant`, …) — see [cli/spec.md](../cli/spec.md)
- **Human observer:** Murrmure Desktop — see [desktop/spec.md](../desktop/spec.md)

Historical normative spec: [archives/superseded/config-shell-v1.md](../../archives/superseded/config-shell-v1.md)

Config HTTP routes remain live; only the Configure **shell mode** was removed.
