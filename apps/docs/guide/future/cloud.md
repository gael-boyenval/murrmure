# Planned: Murrmure Cloud

**Status: not shipped.** Murrmure today is **Desktop + CLI only**.

This page exists so older docs and plans have a single place to point. Do not build integrations against the items below until they are announced as shipped.

## What might ship someday (no promises)

- Hosted Murrmure with online signup and workspace URLs
- Multi-user accounts, SSO, and org billing
- Optional browser-only access without Desktop
- Federated or remote hub deployments as a managed product

## What ships today

| Surface | Docs |
|---------|------|
| **Murrmure Desktop** | [Desktop guide](../desktop) |
| **CLI (`mrmr`)** | [CLI guide](../cli) |

Install Desktop, use `mrmr connection create` for local trust boundaries, and
`mrmr login` for CLI operators. Default hub URL:
`http://127.0.0.1:8787`.
